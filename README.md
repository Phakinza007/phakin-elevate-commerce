# Elevate Commerce — Fashion E-commerce

Full-featured e-commerce backend with product catalog, orders, customer reviews, and admin tools.

**Live Demo:** [View on Portfolio](https://phakinza007.github.io/my-portfolio/)

## Features
- Product catalog with search, category filter, featured items
- Customer registration & JWT auth
- Order creation (guest or logged-in)
- Product reviews & ratings
- Admin order status management

## Tech Stack
**Frontend:** HTML, CSS, JavaScript  
**Backend:** Node.js, Express, SQLite, JWT, bcrypt
